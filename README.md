# Учебное приложение Банк
## Тестовые данные для входа:
- nic: ci, pin: 1111

## Технологии
- HTML
- CSS
- JavaScript

## Функционал 

- Вход в аккаунт
- Баланс счета и текущая дата
- Список операций
- Перевод/займ средств
- Закрытие счета
- Таймер входа
